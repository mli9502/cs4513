To build the project, run "make".
To clean the project, run "make clean".

To run rm, use command "./rm [-f -h -r] file [file ...]"
To run dv, use command "./dv [-h] file [file ...]"
To run dump, use command "./dump [-h]"

The source code for rm, dv and dump are conatained in "rm.c", "dv.c" and "dump.c".
The bash script "projectTest.sh" is used to run the experiments on my computer.
The report is contained in "CS4513project1report.pdf" file.
